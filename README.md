# Forge

**[joinforge.io](https://joinforge.io)** — Where Data + AI Professionals Unite

Forge is Ohio's largest community of data and AI builders. We bring together engineers, researchers, and practitioners to learn, grow, and build together through events, open-source collaboration, and career resources.

## The Story

Forge started as the **Ohio AI Collective** — a grassroots effort to connect the state's growing community of AI and data professionals. As the community grew beyond meetups into ship sessions, lightning talks, a job board, and open-source initiatives, we rebranded to **Forge** to better reflect what we're building: a place where ideas, skills, and careers are forged together.

The name captures our ethos — hands-on, builder-first, community-driven.

## What We Do

- **Events** — Lightning talks, ship sessions, workshops, and more
- **Jobs** — A curated job board for Ohio's AI and data ecosystem
- **Open Source** — Community-driven projects and Ohio-specific datasets
- **Newsletter** — Weekly updates at [joinforge.beehiiv.com](https://joinforge.beehiiv.com/)

## Tech Stack

- React + Vite
- Tailwind CSS + Radix UI
- Framer Motion
- Deployed on GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Data Management

Content is managed through local JSON files in `src/data/`:

- `partners.json` — Partner logos and info
- `jobs.json` — Job listings (markdown descriptions)
- `events.json` — Events (markdown descriptions)

See `CLAUDE.md` for data editing guidelines.
